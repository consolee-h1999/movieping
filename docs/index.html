Movieping

Get notified when films on your Letterboxd watchlist (or a list) are screening at DC-area theatres.

Add this to users.json

v1 signups are manual: copy the entry below into users.json in the GitHub repo (or send it to the repo owner). Your Letterboxd watchlist or list must be public.

How it works

  1. Signups live in users.json in the repo.
  2. GitHub Actions runs daily and scrapes local theatre sites.
  3. Matches are posted to a Discord channel via webhook.
